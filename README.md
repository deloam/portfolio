# Porfolio Game

Este é um jogo simples criado com HTML5 Canvas e JavaScript. O jogo é semelhante ao clássico Asteroids, onde o jogador controla um círculo para coletar itens enquanto evita inimigos.

## Como Jogar

- Use as teclas WASD ou as setas do teclado para mover o jogador.
- Colete os itens verdes para aumentar sua pontuação.
- Evite os inimigos, colidir com eles resultará em um fim de jogo.
- A cada 20 pontos, a cor dos inimigos mudará e a velocidade do jogador aumentará em 5%.

## Controles

- `W` ou `Seta para cima`: Mover para cima
- `A` ou `Seta para a esquerda`: Mover para a esquerda
- `S` ou `Seta para baixo`: Mover para baixo
- `D` ou `Seta para a direita`: Mover para a direita

## Funcionalidades

- O jogo possui uma tela de início que permite ao jogador iniciar o jogo.
- Quando o jogador perde, uma tela de fim de jogo é exibida com a opção de jogar novamente.
- A pontuação do jogador é exibida na tela durante o jogo.
- O fundo do quadro é preto com estrelas brancas para simular o espaço.

## Tecnologias Utilizadas

- HTML5 Canvas: Para renderizar os elementos do jogo.
- JavaScript: Para implementar a lógica do jogo e interações do usuário.

## Como Executar

Basta abrir o arquivo `index.html` em um navegador da web compatível com HTML5.
